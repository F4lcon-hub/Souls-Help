How it works

This site lets you publish and find co-op/help requests for Soulslike games.

  • Post a request by filling the form. By default it expires in 4 hours (you can extend to 24h).
  • Use the top filter to view only one specific game.
  • Each card has actions: Copy, Share, and Remove.
  • Boss/location suggestions for Elden Ring come from the public Elden Ring Fan API.
  • Requests are stored in a backend (API) backed by a Supabase database.
  • No login: requests are public. Do not share sensitive information.

If the top indicator shows "offline", actions fall back to local storage and will sync when the server becomes reachable.

Start local server

To synchronize requests across users, you need to start the local server.

cd server && npm install && npm start
Tip: keep this page open and run the command in a terminal. Then click Check server.
